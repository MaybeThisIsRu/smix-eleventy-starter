# React on Gulp - Starter

## Framework

* React
  * Babel transpilation for ES6 support
  * Module bundling via Browserify
  * `react-router` for routing
  * `axios` for API calls
* PostCSS
  * Includes: imports, nesting, purge, minification
  * Autoprefixer
  * TailwindCSS

## How to Use

### Local Development

* `nvm use`
* `npm install`
* `npm run dev`
* Open `localhost:3000` on your browser

### Production

* `npm run build`
